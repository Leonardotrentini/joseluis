# Pasta de Imagens

Coloque as seguintes imagens nesta pasta:

1. **hero-hombre-mujer.jpg** - Imagem do hero: Hombre mayor sentado com mulher deitada ao fundo (cena emocional)

2. **manos-escribiendo.jpg** - Close-up em preto e branco das mãos envelhecidas escrevendo

3. **manos-entrelacadas.jpg** - Mãos entrelaçadas com pessoa deitada ao fundo (para seção de dor)

4. **libro-foto.jpg** - Livro sobre mesa com foto antiga de casal (opcional, para seção sobre o livro)

5. **fechamento.jpg** - Imagem para seção de fechamento emocional (opcional)

## Formato recomendado:
- JPG ou PNG
- Resolução: mínimo 1200px de largura
- Otimizadas para web (compressão)
